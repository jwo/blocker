# Blocker - Sample Rails Blog app, on docker

* Monitoring: Prometheus
* DB: mysql

## Getting Started

1. Digital Ocean, create docker app
2. sign in to new app
3. `apt-get install git docker-compose`
4. mkdir /var/blocker
5. cd /var/blocker
6. `docker-compose build`
7. `docker-compose up`
